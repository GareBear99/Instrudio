# Production Notes

This repository is the single source of truth for Instrudio web demos and for the future plugin/mobile bridge layers.

## Locked Principles
1. Instrument ids never drift between web, plugin, and mobile.
2. Release manifest is the authority for compatibility.
3. Free web pages remain public demos.
4. Paid access is for connected features, update entitlement, and external MIDI / plugin-linked use.
5. Frontends may differ visually, but the instrument catalog and transport contracts stay shared.
