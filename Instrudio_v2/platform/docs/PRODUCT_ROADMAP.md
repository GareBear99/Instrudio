# Instrudio Product Roadmap

## Phase 1 — Static Suite
- Public web demos
- Shared playback + score import
- GitHub Pages deployment

## Phase 2 — Shared Instrument Authority
- JSON instrument definitions
- Shared catalog and release manifest
- Version compatibility checks across web, plugin, and mobile

## Phase 3 — Plugin Bridge
- Desktop plugin reads same instrument ids and presets
- App-to-plugin transport via bridge contract
- External MIDI mode routing

## Phase 4 — Mobile Subscription
- Free demo browsing
- Paid entitlement for connected play, plugin linkage, and ongoing updates
- Monthly subscription target: $0.99

## Phase 5 — Store / Production
- iOS App Store client
- Desktop plugin packaging
- Managed update endpoint and receipt validation
