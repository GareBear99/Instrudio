Instrudio SSOT Benchmark Suite

Automated evaluation for: "A single-source-of-truth architecture for cross-platform virtual instrument ecosystems"

Test 1 — SSOT Definition Fetch + Apply Timing

Measures remote fetch (GitHub raw), local fallback, cached hit, and definition apply times. Runs N=20 trials per condition.

Test 2 — MIDI Pipeline Latency

Synthesizes N=200 note events through InstrudioBridge.playMidi(), measures dispatch-to-return latency. Note: this measures the JS pipeline, not audio output latency (which depends on AudioContext.baseLatency).

Test 3 — Update Propagation Timing

Calls checkForUpdates() N=10 times, measuring the full round-trip to GitHub for version comparison.
